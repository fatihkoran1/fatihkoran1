## 🚀 Projects

🌍 **Low-Resource Language Fine-Tuning Assistant (In progress)**  
AI writing assistant exploring small language model fine-tuning for a low-resource language, including dataset preparation, demo interface, evaluation and responsible AI topics.  

**Tech:** Python, Hugging Face Transformers, LoRA/QLoRA, Streamlit, Pandas

🧠 **NSE Open Response Analysis with AI (Team project – In progress)**  
AI application that analyzes large amounts of open student survey feedback in a privacy-safe way. The system uses local GenAI and RAG techniques to retrieve, summarize, and visualize recurring themes from student comments, helping Fontys understand student feedback without manually reading thousands of responses.  

**Tech:** Python, RAG, LLM, ChromaDB, Embeddings, Streamlit, Pandas

🍌 **Fruit Quality Classifier**  
AI computer vision application that analyzes banana shelf images to classify overall shelf quality as Fresh, Eatable, Near Expiry, or Spoiled. I manually created and labeled the dataset, trained CNN models, compared them with traditional ML approaches, and used Grad-CAM to explain model decisions. The project was designed around real supermarket workflows and validated through employee interviews. 

**Tech:** Python, CNN, TensorFlow/Keras, OpenCV, Scikit-learn, Pandas, NumPy, Grad-CAM

📊 **IBCS Dashboard Title Analysis (Team project)**  
AI-based system that analyzes dashboard and visualization screenshots to check whether titles follow IBCS Rule B for clear business reporting. The application extracts title text, analyzes the Who-What-When structure, identifies missing or unclear elements, and suggests improved title versions.  

**Tech:** Python, YOLOv8, OCR, LLM, Scikit-learn, Pandas, NumPy

📊 **StackEval Benchmark Recreation**  
AI benchmarking project that compares multiple LLMs on real Stack Overflow-style coding questions. The system uses API-based answer generation, human scoring, and result analysis to evaluate model reliability, correctness, and real-world coding performance.  

**Tech:** Python, OpenAI Anthropic and OpenRouter API, Streamlit, Pandas

🎌 **MyAnimeRate – Anime Review Website**  
Full-stack anime review platform where users can explore anime, rate titles, write, update and delete reviews, and manage their favorite anime list. The project uses a layered architecture with UI, API, service, repository, and database layers to practice maintainable full-stack development.  

**Tech:** Blazor WebAssembly, ASP.NET Core Web API, C#, SQL Server, Entity Framework Core, Bootstrap, Blazored.LocalStorage
